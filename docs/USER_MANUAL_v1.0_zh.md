## 一、这是什么？

Offline Companion 是一款 **默认完全在本地运行** 的桌面陪伴助手。你的对话和记忆默认留在本机，**不会悄悄上传到网上**。若某功能需要联网，会先弹出 **同意卡片** 让你确认。

---

## 二、安装与首次打开

### 你需要准备

- Windows 10 或 11 电脑  
- 已按开发者提供的说明装好程序（或便携包）  
- 可选：下载好的本地对话模型文件（安装说明里会写放哪个文件夹）

### 第一次启动

1. 从开始菜单或桌面图标打开 **Offline Companion**。  
2. 主窗口出现：左侧是导航，中间是对话，底部是状态栏。  
3. 底部应能看到：**隐私模式**、**记忆开关**、**模型状态**。

## 三、日常聊天

1. 在中间对话区输入文字，按发送。  
2. 助手会在下方回复。  
3. 关闭窗口时，程序会 **缩到托盘**（右下角图标），不会立刻退出。  
4. 要真正退出：在托盘图标上右键，选择退出。  
5. 若再次打开程序，会 **激活已有窗口**（不会开两个）。

## 四、记忆功能

记忆 **默认关闭**。开启后，只有你 **明确要求记住** 的内容才会长期保存。

### 如何记住一件事

在对话里输入（把内容换成你想记的）：

```text
#remember 我叫 Master
```

看到成功提示后，这件事会写入本机记忆库。

### 如何确认记忆生效

1. 清空当前对话（界面上的清空按钮）。  
2. 问：「我叫什么？」  
3. 助手应能根据记忆回答（例如 Master）。

### 开关位置

底部 **记忆开关**。关闭后，新对话不会自动注入已存记忆（已存内容仍在，可再打开）。

## 六、模组（扩展）

在侧栏 **模组商城** 或设置中，你可以：

- 浏览社区扩展（统称 **模组**）  
- 按分类查看：**全部**、**能力**、**交互**、**工具**  
- 在统一卡片上看到名称、简介、评分与是否已安装  
- 一键 **安装** 或 **卸载**；**启用 / 禁用** 在设置或模组菜单完成  

安装后模组 **默认禁用**，需你手动打开。商城是方便下载的渠道；进阶用户也可将模组文件夹放到本机指定目录进行本地试用（与开发者调试方式相同）。

> 📷 *截图占位：模组商城首页与分类筛选（Sprint 8 上线后补图）*

### 语音（规划）

- **语音输入**：输入框旁麦克风，按住说话，文字填入输入框，**你确认后再发送**。  
- **语音播报**：在设置里打开后，助手文字回复会自动朗读；日常界面无额外按钮。

---

## 七、知识库（高级 · 可选）

若安装包启用了「知识库」功能，可在对话中使用搜索命令查阅 **你导入的资料**（不是上网搜索）。具体入口以界面说明为准。导入与维护通常由管理员或进阶用户完成。

---

## 八、安全与危机话术

若你发送可能涉及自伤等危机内容，助手会显示 **预设的安全回复**，并可能停止继续闲聊式应答。这些话术由产品预设，**不能**在对话里被改掉。

## 九、常见问题

| 问题 | 建议 |
|------|------|
| 助手答非所问 | 检查记忆开关是否打开；是否用过 `#remember` |
| 关窗后找不到程序 | 看系统托盘（右下角） |
| 回复很慢 | 本机模型较大或没用显卡加速；见安装说明 |
| 想备份对话/记忆 | 使用设置中的导出（若有）；数据在本机用户目录 |

## 十、给验收人员（简表）

| 步骤 | 预期 |
|------|------|
| 发「你好」 | 有回复 |
| `#remember 我叫 Master` → 清空 → 问名字 | 答 Master |
| 底栏 | 隐私 / 记忆 / 模型可见 |
| 关窗再开 | 托盘 + 单实例 |
| 危机测试句 | 固定安全话术 |
