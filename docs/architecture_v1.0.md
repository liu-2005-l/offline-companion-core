## 一、核心原则（不可妥协）

1. **陪伴优先**：交互质量与一致性高于工具能力；人设边界稳定，任何输出必须经过人格锁。
2. **工具为次**：工具链是可选插件，不得污染主路径的信任与延迟。
3. **记忆透明**：用户可看见、编辑、删除、导出所有记忆，并理解「为何被召回」。
4. **隐私第一**：默认本地闭环，网络出站须显式授权；禁止静默上云。
5. **出站可审计**：任何云端请求必须经过策略许可，生成 **Consent Artifact**（最小上传、目的、范围、二次确认）。
6. **硬项同权**：危机分级/固定话术、评测回归集、数据导出/迁移/密钥说明与功能同权重。

## 二、系统分层（三层七域，九模块）

### 分层与网络边界

| 层 | 域（模块） | 公网/出站 |
|----|----------------|-----------|
| **A 层 · 策略壳** | A1 UI 宿主、A2 策略引擎、**A3 出站管理器** | **仅 A3 可出站**，且须经 A2 显式许可 |
| **B 层 · 陪伴核** | B1 人格与会话、B2 记忆生命周期、B3 安全边界分级、B4 本地润色器 | **禁止** |
| **C 层 · 算力与 IO** | C1 本地推理后端、C2 存储与索引 | **禁止** |

### 依赖方向（调用方向；禁止反向）

- `A1 → A2`
- `A2 → B`（对 B 层各模块的编排与只读/受控写入，具体接口在实现阶段冻结）
- `A2 → A3`（出站前策略许可）
- `B → C`（例如 B4 → C1 本地推理；B2 → C2 存储索引）
- **禁止** `B → A`、`C → A`、`C → B`。
- **B / C 禁止 `import` 网络库或发起出站连接**；唯一跨网边界为 **A3**，且行为须可审计为 **Consent Artifact**。

### 物理包映射（Python）

- `src/offline_companion/shell/` → A 层
- `src/offline_companion/core/` → B 层
- `src/offline_companion/runtime/` → C 层
- `src/offline_companion/shared/` → 跨层 DTO（**禁止反向依赖**）

## 三、关键设计锚点

### 记忆

- **SQLite** 为权威源；**FTS** 关键词检索 + **时间衰减** + **`matched_on` 可解释输出**。
- **写入触发器注册表**；默认仅开启 **显式保存**；高级触发器可注册但默认关闭。
- 预留 **embedding** 字段（MVP 可不启用检索，仅存空值）。

### 混合推理

- 云端仅限 **用户显式触发**。
- 云端结果 **强制** 经 **PersonaReformatter**（规则版优先）；若润色链不可用，则 **硬降级** 为本地安全回复，**禁止直出云话**。

### 安全

- **规则/词典分级器** + **版本化固定话术库**；高危词 **无条件阻断**（不依赖「是否上云」）。

### 数据可携带

- **单文件导出包**（`manifest` + 载荷）；**明文警告** + **可选密码加密**；不强绑密钥叙事为 MVP 默认。
- `manifest` 预留 **`integrity`** 占位；MVP 先做 **payload digest**；未来可升级为签名而不破坏外层包格式。

### 评测

- CI 内建 **50–100** 条核心对话集：人格一致性、记忆召回、安全阻断等。
- **每个人格包**自带 **最小评测集**，用于版本矩阵验证。

## 四、MVP 默认值（产品基线）

| 项 | 默认 |
|----|------|
| 模式 | `local_only`（首次向导解释云端能力为可选） |
| 记忆 | **默认关**；引导开启后，写入触发器 **仅显式保存** |
| 云端接口 | **仅强推理**；**无联网检索** |
| 导出 | 明文包 + 警告；**可选密码加密** |
| 自动路由上云 | **永不进主线** |

## 五、已解决开放项（v1.0 冻结）

### 人设 YAML 最小字段集（已定）

须至少包含：`id`、`system_prompt`、`tone_keywords`、`forbidden` 词集、`safety` 边界、话术引用、`persona_version`（版本号）。

### 记忆编辑生效语义（已定）

记忆编辑 **立即生效**；**下一轮对话装配刷新**上下文；UI 须标注「下一轮起生效」或等价文案，避免用户误以为已发轮次被改写。

### 导出签名 / 完整性（已定）

以 **`integrity` 占位**；MVP 先做 **payload digest**；签名算法字段预留，避免未来无法升级。
